Mesa Configuration Tool
=======================

The Mesa Configuration Tool creates LinuxCNC configuration files for
5i25, 6i25, 7i76e, 7i80db, 7i80hd, 7i92, 7i93, 7i95, 7i96, 7i96S, 7i97
and 7i98.
